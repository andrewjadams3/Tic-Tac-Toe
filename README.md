Tic-Tac-Toe
===========

An unbeatable tic-tac-toe game in Ruby.

##Setup

    git clone https://github.com/andrewjadams3/Tic-Tac-Toe.git
    cd Tic-Tac-Toe
    bundle install

##Play

    ruby tic-tac-toe.rb
Use arrow keys to move, press 'Return' to place a piece. Good luck!
